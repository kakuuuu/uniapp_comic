### 使用说明
本项目数据为动态获得，使用[开源有态度的漫画CMS](https://github.com/hiliqi/xiaohuanxiong)作为后端，需先使用该CMS建站,自行上传漫画、文件,配置api_key后,在本项目根目录config.js文件中手动修改新站点的url_config和api_key即可实现该效果。默认使用作者的服务器站点进行请求,获取不稳定。
